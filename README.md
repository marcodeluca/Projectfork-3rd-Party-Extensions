Projectfork-3rd-Party-Extensions
================================

A collection of 3rd Party Extensions for Projectfork 4.


##Quick Start

####Minimum Requirements to install Projectfork 3rd Party Extensions
* Projectfork 4.x
* Joomla 3.x
